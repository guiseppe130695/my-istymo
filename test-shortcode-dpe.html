Test du Shortcode DPE

Utilisez ces shortcodes dans une page WordPress :

  • [dpe_test] - Test simple (doit afficher une boîte jaune)
  • [dpe_panel] - Test complet avec diagnostic
  • [dpe_simple] - Alias pour dpe_panel (même fonctionnalité)

Instructions de test :

  1. Créez une nouvelle page dans WordPress
  2. Ajoutez d'abord [dpe_test] pour vérifier que les shortcodes fonctionnent
  3. Si ça marche, ajoutez [dpe_panel] pour le test complet
  4. Publiez la page
  5. Visitez la page pour voir les résultats

Ce que vous devriez voir :

  • Une boîte rouge avec "TEST SHORTCODE DPE"
  • Des vérifications en vert/rouge pour chaque composant
  • Le template DPE complet si tout fonctionne

Si vous ne voyez rien :

  • Le shortcode n'est pas enregistré
  • Il y a une erreur PHP
  • Le plugin n'est pas activé
